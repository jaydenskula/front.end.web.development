ENTRY 1 | CLASS PORTAL

I then went on to formatting my assignments, projects, and process blog page. ENTRY 2 | ASSIGNMENT 1

The first line of code was styled in Javascript through the DOM. This completes the first requirement of manipulating a css property through the DOM. I killed two birds with one stone in the second line of code. I added an event listener for a mouse click. When the text is clicked on, the text presented changes. This is not a CSS property and it is one type of event of two for this assignment. The final line has event listeners for when the mouse hovers over and off the text. When the mouse is hovering over the text, the text changes color. It reverts back to the original color if the mouse is not hovering over it. Overall, this assignment was fairly simple. I had a few challenges because it has been a few months since I have used Javascript, so I simply had to refresh my memory.
