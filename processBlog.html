ENTRY 2 | ASSIGNMENT 1

The first line of code was styled in Javascript through the DOM. This completes the first requirement of manipulating a css property through the DOM. I killed two birds with one stone in the second line of code. I added an event listener for a mouse click. When the text is clicked on, the text presented changes. ENTRY 3 | ASSIGNMENT 2

Present in this assignment: - Input type "submit" - Input type "number" - Input type "datetime-local" - Input type "color" - 'select' and 'option' This was another simple assignment. I had a bit of trouble figuring out the Javascript syntax so that it takes the user input and manipulates the page directly based on the input, however with some time I figured it out. I was not too worried about the aesthetics/style of this webpage as I was more focused on meeting all the requirements. The first form lets the user enter a number 1-100. Upon clicking submit, the webpage alerts the user with the number they entered. The second form has the datetime-local input type. The date and the time that the user clicks is displayed in the console. The third form gives the user a list of colors that they could select. Their selection changes the color of the text on the page. The last form is a color picker which manipulates the background color of the page.
